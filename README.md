# All In One

A simple static landing page that brings India’s top shopping and grocery platforms together in one place.

## Overview

`All In One` is a single-page web application built with HTML, CSS, and JavaScript. It provides quick access to popular retail and grocery services using a clean, modern design with category cards and a contact section.

## Features

- Shopping and grocery categories with direct platform links
- Interactive card hover overlays for an engaging UI
- Sticky navigation for fast section access
- Contact form with simulated submission feedback
- Scroll animations for card entrance effects
- Responsive layout for desktop and mobile screens

## How to Use

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Click any card to visit the corresponding service.
4. Use the contact form to simulate sending a message.

## Technologies

- HTML5
- CSS3
- JavaScript
- Google Fonts (`Outfit`)

## Notes

- This is a static frontend project and does not include a backend.
- The contact form is mocked in `script.js` and uses a timeout to simulate submission.
