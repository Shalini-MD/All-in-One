# All In One - Shopping & Grocery Hub

Welcome to All In One! This is a single-page web application designed to be your ultimate shopping and grocery hub. It provides a centralized, easy-to-use platform to access your favorite online retail and grocery delivery services in India.

## Features

1. Categorized Hub:  Quick access divided into distinct categories:
  - Shopping: Amazon, Meesho, Myntra, Flipkart.
  - Groceries: BigBasket, Blinkit, Zepto, JioMart.
2. Responsive Design: A clean, modern UI styled with custom CSS (`task1.css`).
3. Smooth Navigation: Sticky navigation bar for quick jumps to "Home", "Categories", and "Contact".
4. Contact Form: An integrated query section to send messages and feedback directly to the support team.
- **Custom Typography**: Built with modern "Outfit" fonts for a premium look and feel.

## How to Run

Since this is a static HTML/CSS website, no complex setup or server is required:
1. Clone or download the repository to your local machine.
2. Navigate to the `Task1` directory.
3. Open `task1.html` in any modern web browser (Chrome, Firefox, Edge, Safari).

## Technologies Used

1. HTML5: For semantic structure.
2. CSS3: For custom styling, flexbox layouts, hover animations, and responsiveness.
3. Google Fonts: Using the "Outfit" font family.
